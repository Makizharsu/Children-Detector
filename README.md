# Children-Detector
Detecting whether they are child or not using machine learning.

# To Run this

1) Clone the repositary

2) Run the gui.py file
   
3) Upload the image
 
4) Upload the cropped image of face to get accurate results
  
5) Detect the image
 
6) view the results
